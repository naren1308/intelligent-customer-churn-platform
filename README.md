# Intelligent Customer Churn + Revenue Retention Platform

An end-to-end machine learning project to predict customer churn, estimate revenue impact, and suggest actionable retention strategies.

## Project Structure
- `data/`: Raw and processed data.
- `notebooks/`: Jupyter notebooks for EDA and modeling.
- `src/`: Reusable Python modules for preprocessing, training, and predicting.
- `dashboard/`: Streamlit application.
- `models/`: Saved model artifacts.

## Tech Stack
- Python, pandas, numpy, scikit-learn, XGBoost
- SHAP for Explainable AI
- Streamlit for the interactive dashboard
